## ğŸ“ Support & Help

### Common Questions

**Q: How do I change the company name?**  
A: Search for "LogiConnect" in all HTML files and replace with your company name.

**Q: How do I add images?**  
A: Save images to `assets/images/` folder and update the `src` attributes in HTML.

**Q: Does the contact form send emails?**  
A: No, this is a demo. To enable emails, you need a backend server or service like Formspree.

**Q: How do I test on my phone?**  
A: Start server with: `python -m http.server --bind 0.0.0.0 8000` then visit your IP address.

**Q: Can I change the colors?**  
A: Yes! Edit CSS variables at the top of `css/styles.css`.
